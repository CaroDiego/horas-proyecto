# ⏳ Gestor de Horas Proyecto  
🏗️ *En desarrollo*  

Aplicación para gestionar y visualizar las horas trabajadas en un proyecto, así como el tiempo restante.  

🔗 **Página web:** [Gestor de Horas](https://gestorhorasproyecto.vercel.app/)  

## 🚀 Tecnologías utilizadas  
- ![Vite](https://img.shields.io/static/v1?label=&message=Vite&color=646CFF&logo=vite&logoColor=white)  - Entorno de desarrollo rápido para React  
- ![React](https://img.shields.io/static/v1?label=&message=React&color=61DAFB&logo=react&logoColor=white)   - Biblioteca para la creación de interfaces de usuario  
- ![Supabase](https://img.shields.io/static/v1?label=&message=Supabase&color=3ECF8E&logo=supabase&logoColor=white)  - Backend como servicio basado en PostgreSQL  

## 📌 Características  
✔️ Registro y seguimiento de horas trabajadas  
✔️ Cálculo de horas restantes  
✔️ Previsión de finalización de proyecto

## 📦 Instalación y ejecución  

1. Clona el repositorio:  
   ```bash
   git clone https://github.com/tu-usuario/gestor-horas-proyecto.git
   cd gestor-horas-proyecto
   ```
2. Instala las dependencias:  
   ```bash
   npm install
   ```
3. Inicia el proyecto:  
   ```bash
   npm run dev
   ```
